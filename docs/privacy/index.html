2. 収集する情報

本アプリが収集する情報は以下のとおりです。

【端末識別子（device_id）】
アカウント登録は不要です。ユーザーを識別するため、端末固有のIDを匿名で生成・保存します。氏名・メールアドレス等の個人情報は一切収集しません。

【学習データ】
単語の表示回数・正解数・不正解数・学習状況・習得日時を、お使いの端末内に保存します。これらのデータはサーバーには送信されず、個人を特定できる情報も含みません。

【課金情報】
有料プランの購入情報はApple App StoreおよびRevenueCatによって管理されます。カード情報等の決済情報は運営者のサーバーには保存されません。

3. 情報の利用目的

収集した情報は以下の目的のみに使用します。

・学習進捗の管理・通知スケジュールの最適化
・アプリの品質改善

4. 第三者提供

収集した情報を第三者に販売・提供することはありません。ただし以下のサービスを利用しています。

・Supabase（単語データの配信）
　https://supabase.com/privacy
・RevenueCat（課金管理）
　https://www.revenuecat.com/privacy
・Apple App Store（アプリ配信・決済）
　https://www.apple.com/legal/privacy/

5. 広告について

現時点では広告を表示していません。将来的に広告を導入する場合は、本ポリシーを更新した上で事前にお知らせします。

6. データの保管・削除

本アプリにはアカウント登録がなく、学習データはお使いの端末内にのみ保存されます。すべてのデータを削除したい場合は、本アプリをアンインストール（削除）してください。アンインストールすると、端末内に保存された学習データはすべて削除されます。
